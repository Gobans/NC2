# MenuCatcher
메뉴판을 Live Text 로 인식하여 메뉴 이름을 가져온 후 영양소에 대한 정보를 가져와 보여주는 앱

<br>

## 🍎 Appstore Link
- https://apps.apple.com/app/메뉴캐처/id6444089244

<br>

## 📌 Features
- 메뉴판 내의 음식 이름 인식 (Live Text)
- 음식 이름을 카테고리 별로 나누어 DB 데이터와 대조 (ML)
- 인식된 음식 데이터를 사용자에게 표시

<br>
